# 김비서 — 업무 대시보드

마케팅팀을 위한 통합 업무 관리 대시보드. 오늘의 할 일, 주간 일정, 매출 분석, 프로젝트 진행률을 한눈에 확인하세요.

---

## 🎯 주요 기능

### 📋 오늘 할 일
- 우선순위별 할 일 관리 (높음/중간/낮음)
- 실시간 추가/삭제 가능
- 체크박스로 완료 표시 (취소선 처리)
- 담당자 및 마감일 표시

### 📅 이번 주 일정
- 7일 일정 캘린더 (월~일)
- 주요 이벤트 태그 표시
- 오늘 강조 표시

### 💰 매출 현황 요약
- 총 매출액 및 거래 현황
- 제품별 매출 비중 (도넛 차트)
- 제품별 판매 트렌드 (라인 차트)
- 지역별 매출 분석

### 📊 프로젝트 진행률
- 5개 주요 프로젝트 현황
- 진행률 바 시각화
- 프로젝트별 담당자 및 마감일

### 🌤️ 실시간 날씨
- 브라우저 위치 기반 자동 감지
- 현재 온도, 날씨, 체감온도
- 습도, 풍속 정보

## 📁 파일 구조

| 파일 | 역할 |
|------|------|
| **업무대시보드.html** | 메인 대시보드 (반응형, 다크모드) |
| **매출보고서.html** | 1~2월 매출 데이터 분석 보고서 |
| **subtrac_분석보고서.html** | SubTrackr 사이트 분석 보고서 |
| **마케팅업무흐름.svg** | 마케팅 업무 프로세스 다이어그램 |
| **CLAUDE.md** | 프로젝트 문서 |

---

## 🛠️ 사용 기술

### Frontend
- **HTML5** — 시맨틱 마크업, 폼 입력
- **CSS3** — Grid/Flexbox 반응형 레이아웃, 다크모드 변수 시스템
- **JavaScript** — DOM 조작, 이벤트 처리, 비동기 API 호출

### 외부 API
- **Open-Meteo** — 실시간 날씨 데이터 (무료, API 키 불필요)
- **Nominatim** — 역 지오코딩으로 도시명 취득

### 차트
- **Chart.js** — 도넛차트, 라인차트, 막대차트 렌더링
